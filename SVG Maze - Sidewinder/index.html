How does the algorithm work?
  • Visit every cell, traversing each row from the left to the right
  • Remove a gate east or one of the visited, opened cells' north
  • Do not create any exit

East or what?!

As you traverse the grid, one row at a time and from the left to the right, you select a gate east or north.

If east

  • Remove the gate

If north

  • Consider the cell with the cells coming earlier in the row
  • Pick one at random
  • Remove the gate
  • Close the cells. This precludes them from being modified in the same row
